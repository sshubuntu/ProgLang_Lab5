## Lab5

Консольное приложение на .NET 8, парсит арифметические выражения (TDOP/Pratt), генерирует IL через `System.Reflection.Emit` и вычисляет в `long` с выводом промежуточных подвыражений.


### Запуск
```powershell
cd Lab5/ExpressionEvaluator
dotnet run
```
Если увидите ошибку вида “файл ...ExpressionEvaluator.exe занят”, закройте ранее запущенный экземпляр программы и повторите `dotnet run`.

### Команды
- `expr <выражение>` — разобрать и сгенерировать функцию для выражения.
- `set <имя> <целое>` — задать значение переменной.
- `do` — выполнить выражение, вывести все подвыражения и ответ.
- `exit` — выйти.
